# 📖 User Manual (Chrome Extension)

## 🧩 AI Email Reply Generator Extension

This Chrome extension helps users generate AI-powered email replies directly inside Gmail using Google Gemini AI.

---

## 🚀 How to Install Extension

### 1️⃣ Download the Project

* Clone or download the repository:

```bash
git clone https://github.com/your-username/ai-email-reply-generator.git
```

OR download ZIP and extract it.

---

### 2️⃣ Open Chrome Extensions

* Open Google Chrome
* Go to:

```
chrome://extensions/
```

---

### 3️⃣ Enable Developer Mode

* Turn ON **Developer Mode** (top-right corner)

---

### 4️⃣ Load Extension

* Click **“Load Unpacked”**
* Select the `extension/` folder from the project

---

## 📩 How to Use

### 1️⃣ Open Gmail

* Go to [https://mail.google.com](https://mail.google.com)
* Open **Compose Email** window

---

### 2️⃣ Generate AI Reply

* Click the **“AI Reply”** button injected inside Gmail toolbar
* Wait while AI processes the email content

---

### 3️⃣ Get AI Response

* The extension will:

  * Read email content automatically
  * Send it to backend API
  * Generate AI reply using Gemini AI
  * Insert response directly into email box

---

### 4️⃣ Edit & Send

* You can edit the generated reply
* Then send email normally from Gmail

---

## ⚙️ Important Notes

* Extension works only on **Gmail (mail.google.com)**
* Internet connection is required (for API calls)
* Backend server must be running or deployed
* If button doesn’t appear, refresh Gmail page
